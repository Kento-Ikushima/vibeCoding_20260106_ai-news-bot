# ⚠️ セキュリティ通知

## 重要な注意事項

このリポジトリは**パブリックリポジトリ**です。以下の情報を**絶対にコミットしないでください**：

- APIキー（Gemini、OpenAIなど）
- アクセストークン（LINE Channel Access Tokenなど）
- シークレット（Channel Secretなど）
- ユーザーIDなどの個人情報

## 機密情報の管理方法

### ✅ 正しい方法

1. **GitHub Secretsを使用**
   - リポジトリの「Settings」→「Secrets and variables」→「Actions」
   - すべての機密情報はここに設定してください

2. **環境変数を使用**
   - ローカル実行時は環境変数を使用
   - `.env`ファイルを使用する場合、`.gitignore`に追加してください

### ❌ 間違った方法

- コードに直接書き込む
- ドキュメント（.mdファイル）に実際の値を記載する
- コメントに機密情報を書く

## もし機密情報を誤ってコミットしてしまった場合

1. **すぐに機密情報を無効化・再発行する**
   - LINE Channel Access Token: LINE Developers Consoleで再発行
   - Gemini API Key: Google AI Studioで再発行

2. **Git履歴から削除する**
   - 方法A: リポジトリを削除して再作成（最も簡単）
   - 方法B: `git filter-branch`またはBFG Repo-Cleanerを使用

3. **コードから削除して再コミット**

## 現在の状態

このリポジトリには機密情報は含まれていません。すべての機密情報はプレースホルダーまたは説明文に置き換えられています。
